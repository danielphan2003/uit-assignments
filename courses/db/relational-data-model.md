# Định nghĩa

- Lược đồ quan hệ bậc $n$: $R(A_1, A_2, ..., A_n)$

- Tập thuộc tính của quan hệ $R$: $R^+ = \{A_1, A_2, ..., A_n\}$
  
  Ví dụ: `HOCVIEN(Mahv, Hoten, Ngsinh, Gioitinh, Noisinh, 
Malop)`

- Tên quan hệ: $R$, $S$, $P$, $Q$

- Thể hiện của quan hệ: $T_R$, $T_S$, $T_P$, $T_Q$

- Bộ/mẫu tin của quan hệ $R(A_1, A_2, ..., A_n)$: $t = (a_1, a_2, ..., a_n), \forall a_i \in Dom(A_i)$

- Miền giá trị của thuộc tính $A$: $Dom(A)$ hay $MGT(A)$

- Giá trị tại thuộc tính $A$ của bộ thứ $t$: $t.A$ hay $t[A]$
